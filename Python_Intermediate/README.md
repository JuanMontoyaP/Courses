# Python Intermediate Course

In this Platzi Course I learned the following topics:

- Supers lists and super dictionaries.
- List and Dictionary comprehensions.
- Anonymous and high order functions.

    - Map function
    - Filter function
    - Reduce function
- Debugging
- Error exceptions.
- Assert statement.
- File management.
